# 3DoF Inverse Kinematic Simulator

Simulator interaktif untuk menghitung dan memvisualisasikan **Inverse Kinematics** pada lengan robot dengan 3 Degrees of Freedom (3DoF). Proyek ini dirancang untuk membantu memahami bagaimana koordinat target ($x, y, z$) diterjemahkan menjadi sudut sendi robot.
